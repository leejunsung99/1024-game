## Game Description

1024 is a simple and addictive puzzle game inspired by number-merging mechanics.  
The goal of the game is to combine tiles with the same numbers to create the **1024 tile**.  
As the numbers grow, the board fills up, requiring strategic thinking and careful moves.

---

## How to Play

- **On Computer**:  
  Use the **arrow keys (↑ ↓ ← →)** on your keyboard to move the tiles.

- **On Mobile**:  
  **Swipe (drag)** in the direction you want the tiles to move.

Tiles with the same number will merge when they collide, forming a new tile with double the value.
